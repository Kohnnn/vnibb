# VNIBB UI Package

> **Package:** `@vnibb/ui`  
> **Framework:** React 18 + Tailwind CSS

---

## Purpose

Design system and UI primitives for VNIBB applications.

> **Status: Intentional future-extraction stub.**
> This package is kept in the workspace as a placeholder for a future
> design-system extraction. It is not imported by `apps/web` today.
> Do not add production dependencies on it until real components are
> extracted here.

---

## Components

- `Button`, `Input`, `Select`
- `Card`, `Modal`, `Dropdown`
- `Table`, `DataGrid`
- `Tabs`, `Tooltip`
- `Loading`, `Error`, `Empty`

---

## Design Tokens

- Colors (dark mode first)
- Typography
- Spacing
- Shadows
- Border radius

---

## Usage

```tsx
import { Button, Card, Table } from '@vnibb/ui';
```
