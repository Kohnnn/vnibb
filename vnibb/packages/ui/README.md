# VNIBB UI Package

> **Package:** `@vnibb/ui`  
> **Framework:** React 18 + Tailwind CSS

---

## Purpose

Design system and UI primitives for VNIBB applications.

---

## Components

- `Button`, `Input`, `Select`
- `Card`, `Modal`, `Dropdown`
- `Table`, `DataGrid`
- `Tabs`, `Tooltip`
- `Loading`, `Error`, `Empty`

---

## Design Tokens

- Colors (dark mode first)
- Typography
- Spacing
- Shadows
- Border radius

---

## Usage

```tsx
import { Button, Card, Table } from '@vnibb/ui';
```
